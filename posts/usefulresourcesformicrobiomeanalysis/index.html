Useful resources for data analysis.

Microbiomes: DADA2

I like to analyse all of my microbiome data directly in R with the Benjamin Callahan's DADA2 package. This is an Amplicon Sequence Variants (ASV) approach, relying on fancy run-specific error models for denoising. The DADA2 tutorial is a great place to start if you're interested in this method.

Metagenomes and MAGs: Anvio

Meren and friends have built an immense resource for analysing metagenomes and metagenome-assembled genomes (MAGs). Anvio has a brilliant interactive interface that can be used to manually refine bins but anvio also wraps a whole suite of tools for functional assessments, phylogenomics, etc. Their webpages include a number of highly detailed tutorials to help you think about your own data.

Eukaryotic Phylogenomics: Phylofisher

Phylofisher is a very handy suite of scripts and tools for eukaryotic phylogenomics. It involves manually inspecting single-gene trees prior to concatenation. Using their precompiled database, you can generate a set of 240 single-gene trees which include 304 reference eukaryotes. To remove contaminants or identity paralogs in single-gene trees, they also provide a very nice application called Parasorter.

Happy Belly Bioinformatics

An excellent resource from Mike Lee that I often give to students who are new to bioinformatics. This also includes tutorials for amplicon analysis and genomics using some of the methods above.

PR2

A wonderfully curated database for eukaryotic SSU analysis. I'd also recommend the pr2 R package listed below.
